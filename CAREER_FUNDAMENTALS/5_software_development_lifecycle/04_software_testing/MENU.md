# Software Testing

Relies on the techniques and methodologies used to evaluate software functionality, performance, and reliability. This subject covers test levels such as unit, integration, system, and acceptance testing, as well as test case design, automated testing, and defect management. Students learn how to identify and report bugs, validate requirements, and ensure that software behaves as expected under various conditions, contributing to the delivery of high-quality, dependable applications.
